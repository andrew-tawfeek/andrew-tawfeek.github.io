neural-network-elements

Neural Network Elements

An interactive web-based tool for analyzing the geometry of ReLU neural networks. Computes polyhedral decompositions of the input space, constructs dual graphs of activation regions, performs spectral analysis via Laplacian eigendecomposition, and visualizes decision boundaries — all in the browser, with no server required. Approximately 4,200 lines of code, released by the Georgia Tech Research Institute.

KnotAI

KnotAI

A PyTorch feedforward neural network classifier for recognizing and classifying knots from their matrix (mosaic) representations. Built as part of the Washington eXperimental Mathematics Lab (WXML) project "Teaching a Computer to Knot," this model encodes knot diagrams as structured matrices and trains a graph neural network to predict knot invariants and knot types.

discrete-gradients

Discrete Gradients

A SageMath library for computing discrete gradient vector fields on simplicial complexes via Morse theory. Implements algorithms for constructing and analyzing discrete Morse functions, computing Laplacian eigendecompositions, and extracting topological features. Released as companion code for the Annals of Combinatorics publication (2023).
